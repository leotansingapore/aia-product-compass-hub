## Create Your Task List

Create your task list and set up these sections:

![Creating task list](https://assets.skool.com/f/8fc2f6045ea549398401f3cdbcc90878/9ad7bc7c16b3418fac7dc584cd15a729b5f33c4e9f5049b6954537d04c63f1f1.gif)

### Task Management Tips

- **Key in all your tasks** — set aside 15 minutes to do a "brain scan" and dump out everything (ad hoc and recurring)
- **Create multiple task lists** if needed (e.g., one for work, one for personal)
- **Create tasks from messages** — have an automatic habit to create tasks whenever you see something important on Lark:

![Creating tasks from messages](https://assets.skool.com/f/8fc2f6045ea549398401f3cdbcc90878/31bc18a98c214158b09a1422745c235c747b9a1927e2455eb74a5e6608393d85.gif)

> **Key principle:** Capture constantly, then set aside one time in the day to process everything properly. Some tasks can't be done now, but the most important thing is to capture everything first, and sort/process them later. This is critical for working on multiple things in parallel.
