# Week 5 — Operating Rhythms & Daily Discipline

**The week's thesis:** Discipline isn't willpower. Discipline is having the right rhythms (weekly team meetings, scorecards, training) and the right math (Time Value of Money, compounding, DCA) — so that showing up correctly is easier than drifting.
